1. System Overview and Thermodynamic Framework
The ECLIPSE biosensor operates through a three-state thermodynamic cascade. Each transition has an
associated free energy that determines whether the sensor will produce signal.
State 1: Cage closed (latch folded, SmBiT sequestered)
         |
         |  Step 1: PfLDH binds binder in latch
         |  dG_binding = -12.0 kcal/mol (v1.0)  /  -16.6 kcal/mol (v2.2)
         v
State 2: Latch open (PfLDH-binder complex, SmBiT exposed)
         |
         |  Step 2: lucKey binds exposed SmBiT
         |  dG_lucKey = -2.32 kcal/mol
         v
State 3: lucKey bound (NanoLuciferase reconstituted -> LIGHT)
The total cascade free energy is the sum of all steps. All must be negative (thermodynamically favorable)
for the sensor to produce signal. The latch opening has a cost (+4.09 kcal/mol) because the latch is
designed to be closed by default — the binder-PfLDH binding energy must overcome this cost.
Net equation for latch opening:
  dG_net = dG_latch_cost + dG_binding
  dG_latch_cost = -RT * ln(K_open) = -0.592 * ln(1e-3) = +4.09 kcal/mol
  v1.0: dG_net = +4.09 + (-12.0) = -7.91 kcal/mol  -->  FAVORABLE
  v2.2: dG_net = +4.09 + (-16.6) = -12.51 kcal/mol  -->  FAVORABLE
Full cascade:
  dG_lucKey = -RT * ln([lucKey]/K_CK) = -0.592 * ln(50) = -2.32 kcal/mol
  v1.0 total: -7.91 + (-2.32) = -10.23 kcal/mol
  v2.2 total: -12.51 + (-2.32) = -14.83 kcal/mol

3. The K_CK Problem — Charge Optimization Thermodynamics
The original RFDiffusion binder LISDAELEAIFAEELDC has 6 charged D/E residues (positions 4, 6, 8, 13,
14, 16) that sit at the lucKey-cage interface. This creates a thermodynamic penalty that collapses K_CK
and eliminates all signal regardless of how well PfLDH binds.
Calculation:
Each charged residue at the lucKey interface imposes ~0.8 kcal/mol electrostatic penalty
Penalty total = 6 residues x 0.8 kcal/mol = 4.8 kcal/mol
Original K_CK: 1e-8 M
dG_CK (original) = -RT * ln(1/K_CK) = -0.592 * ln(1e8) = -10.9 kcal/mol
After grafting original binder:
dG_CK (grafted)  = -10.9 + 4.8 = -6.1 kcal/mol
K_CK (grafted)   = exp(6.1 / 0.592) = exp(10.3) = 2.97e-5  ~  3e-12 M
  -> K_CK drops from 1e-8 to 3e-12 (3000x weaker)
Effect on signal:
  Signal = (latch opens) x (lucKey binds SmBiT)
  With K_CK = 3e-12:
  [lucKey]/K_CK = 500e-9 / 3e-12 = 166,667
  f_base = K_open / (1 + K_open + [lucKey]/K_CK)
         = 1e-3  / (1 + 0.001  + 166,667)
         = 1e-3  / 166,668  =  6e-9  (essentially zero)
  Even if latch opens (theta=1, pull=100):
  Koe = K_open * (1 + 100) = 0.101
  f_signal = 0.101 / (1 + 0.101 + 166,667) = 6.06e-7
  FC = f_signal / f_base = 6.06e-7 / 6e-9 ~ 101  (still very low absolute signal)
  CORE PROBLEM: absolute luminescence is near-zero because lucKey cannot bind.
  The sensor is structurally intact but functionally dead.
Solution — D/E to Alanine mutations:
Preserve PfLDH interface residues: positions 1, 2, 11, 12, 15
Mutate remaining D/E to Alanine:   positions 4, 6, 8, 13, 14, 16
Mutations: D4A, E6A, E8A, E13A, E14A, D16A
Original:  LISDAELEAIFAEELDC  (6 D/E at lucKey interface)
Optimized: LISAAALAAIFAAALAC  (0 D/E at lucKey interface)
Penalty removed: 6 x 0.8 = 4.8 kcal/mol -> 0
K_CK restored:  ~1e-8 M

Script 4 — DDG and Kd improvement calculation
import math
RT = 0.592  # kcal/mol at 37 C
dG_original = -26.3  # kcal/mol
dG_v10      = -12.0
dG_v22      = -16.6
# Kd from Boltzmann (v1.0 anchored to 100 pM from MD)
Kd_v10 = 100e-12
Kd_v22 = Kd_v10 * math.exp((dG_v22 - dG_v10) / RT)
# Kd improvement
ddG_opt_vs_orig = dG_v10 - dG_original   # cost of optimization
ddG_v22_vs_v10  = dG_v22 - dG_v10        # gain from avidity

6. Free Energy of Latch Opening
The core thermodynamic question: does PfLDH binding provide enough energy to open the latch against
its natural closed-state stability?
Script 6 — Full latch opening and cascade analysis

print(f"   dG = -RT*ln(K_open) = -{RT}*ln({K_open}) = +{dG_open_cost:.2f} kcal/mol")
print()
# Binder-PfLDH binding energies from ADCP
dG_v10 = -12.0
dG_v22 = -16.6
print(f"2. BINDER-PfLDH BINDING (from ADCP docking)")
print(f"   v1.0: {dG_v10:.1f} kcal/mol")
print(f"   v2.2: {dG_v22:.1f} kcal/mol")

# Net free energy of latch opening
print(f"3. NET FREE ENERGY OF LATCH OPENING")
for label, dG_b in [("v1.0", dG_v10), ("v2.2", dG_v22)]:
    dG_net = dG_open_cost + dG_b
    K_eq   = math.exp(-dG_net / RT)

dG_lucKey = -RT * math.log(lucKey / K_CK)
print(f"4. lucKey CASCADE STEP")
print(f"   [lucKey]/K_CK = {lucKey:.0e}/{K_CK:.0e} = {lucKey/K_CK:.0f}")
print(f"   dG = -RT*ln([lucKey]/K_CK) = -{RT}*ln({lucKey/K_CK:.0f}) = {dG_lucKey:.2f} kcal/mol")

# Full cascade

for label, dG_b in [("v1.0", dG_v10), ("v2.2", dG_v22)]:

    cascade = dG_net + dG_lucKey
    print(f"   {label}: {dG_net:.2f} + ({dG_lucKey:.2f}) = {cascade:.2f} kcal/mol")

7. LOCKR Fold-Change Model
The LOCKR system is modeled as a three-state equilibrium. Fold-change is the ratio of signal-state to
baseline-state luciferase reconstitution.
Core equations:
theta = [PfLDH] / ([PfLDH] + Kd)                    (fraction binder occupied)
K_open_eff = K_open * (1 + pull * theta)              (effective latch opening w/ PfLDH)
f_base   = K_open     / (1 + K_open     + [lucKey]/K_CK)  (baseline fraction open)
f_signal = K_open_eff / (1 + K_open_eff + [lucKey]/K_CK)  (signal fraction open)
FC = f_signal / f_base
Substituting numbers:
  [lucKey]/K_CK = 500nM/10nM = 50
  f_base = 1e-3 / (1 + 0.001 + 50) = 1e-3/51.001 = 1.96e-5
  At pull=10, theta=1 (saturating PfLDH):
  K_open_eff = 1e-3 * (1 + 10*1) = 1.1e-2
  f_signal = 1.1e-2 / (1 + 0.011 + 50) = 1.1e-2/51.011 = 2.16e-4
  FC = 2.16e-4 / 1.96e-5 = 11.0x
Script 7 — Thermodynamic model: all variants and pull strengths
import numpy as np
K_open = 1e-3
K_CK   = 1e-8
lucKey = 500e-9
def fold_change(pfldh, Kd, pull):
    theta    = pfldh / (pfldh + Kd)
    Koe      = K_open * (1 + pull * theta)
    f_base   = K_open / (1 + K_open + lucKey/K_CK)
    f_signal = Koe    / (1 + Koe    + lucKey/K_CK)
    return f_signal / f_base
def max_fc_ec50(Kd, pull):
    pfldh = np.logspace(-14, -5, 500)
    fcs   = [fold_change(c, Kd, pull) for c in pfldh]
    mfc   = max(fcs)
    half  = (mfc + 1) / 2
    ec50  = pfldh[np.argmin(np.abs(np.array(fcs) - half))]
    return mfc, ec50
Kd_v10 = 100e-12     # 100 pM
Kd_v22 = 42.21e-15   # 42.21 fM

print("-" * 65)
for label, Kd in [("v1.0 (Kd=100pM)", Kd_v10), ("v2.2 (Kd=42fM)", Kd_v22)]:
    for pull in [10, 20, 30]:
        mfc, ec50 = max_fc_ec50(Kd, pull)
        lod = ec50 * 0.1

8. lucKey Competition Analysis — Why Latch Mutations Don't Help
When we attempted to create a v1.1 variant with latch mutations I346S + I356S (analogous to the Baker
Lab's V223S/I238S), the thermodynamic model predicted no meaningful improvement. This section
shows the math.
The denominator problem:
f_base = K_open / (1 + K_open + [lucKey]/K_CK)
Numerically:
  K_open       = 1e-3  = 0.001
  [lucKey]/K_CK = 500nM/10nM = 50
  f_base = 0.001 / (1 + 0.001 + 50)
         = 0.001 / 51.001
  The 50 term DOMINATES. K_open = 0.001 is negligible.
Even with v1.1 moderate estimate (K_open = 2.93e-2, 30x increase):
  f_base = 0.0293 / (1 + 0.0293 + 50)
         = 0.0293 / 51.029
         = 5.74e-4   (only 29x larger f_base)
  For FC calculation, f_base and f_signal both scale similarly
  -> ratio (FC) barely changes
Conclusion: [lucKey]/K_CK = 50 >> K_open in all scenarios.
The fold-change ceiling is set by lucKey competition, not latch stability.

10. Wet Lab: Measuring Pull Strength from Luminescence Data
Pull strength is the one thermodynamic parameter that cannot be determined computationally — it
measures how efficiently the mechanical force of binder-PfLDH binding transfers to latch opening
(allosteric coupling). It is extracted by fitting the experimental luminescence curve.
Method: curve fitting to the fold-change model
from scipy.optimize import curve_fit
import numpy as np
# Fixed constants (from literature + ADCP)
K_open = 1e-3
K_CK   = 1e-8
lucKey = 500e-9
Kd     = 100e-12   # v1.0;  use 42.21e-15 for v2.2
def fold_change_model(pfldh_conc, pull_strength):
    theta    = pfldh_conc / (pfldh_conc + Kd)
    Koe      = K_open * (1 + pull_strength * theta)
    f_base   = K_open / (1 + K_open + lucKey/K_CK)
    f_signal = Koe    / (1 + Koe    + lucKey/K_CK)
    return f_signal / f_base
# pfldh_conc = array of PfLDH concentrations tested (in M)
# fc_measured = array of measured fold-changes at each concentration
pfldh_conc = np.array([1e-12, 10e-12, 100e-12, 1e-9, 10e-9, 100e-9, 1e-6])
fc_measured = np.array([...])   # fill in from experiment
popt, pcov = curve_fit(
    fold_change_model,
    pfldh_conc,
    fc_measured,
    p0=[10],          # initial guess: pull=10
    bounds=(0, 100)   # physically reasonable range
)
pull_strength = popt[0]
pull_std      = np.sqrt(pcov[0,0])
print(f"Pull strength = {pull_strength:.1f} +/- {pull_std:.1f}")
print(f"Expected fold-change at pull={pull_strength:.0f}: {1+pull_strength:.0f}x")
